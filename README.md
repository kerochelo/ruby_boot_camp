## Ruby Boot Camp
### setup
0. clone
```sh
git clone git@github.com:kerochelo/ruby_boot_camp.git
```

0.5. cd
```sh
cd ruby_boot_camp
```

1. build
```sh
docker compose build
```

2. container run
```sh
docker compose up
```

3. container access
```sh
docker compose exec ruby ash
```

4. exec ruby or irb
- change dir
```sh
cd app
```
- exec 
```sh
ruby hoge.rb
```
or
```sh
irb
```

Good Luck!
