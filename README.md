## Ruby Boot Camp
### setup
0. clone
```sh
$ git clone git@github.com:kerochelo/ruby_boot_camp.git
```

1. build
```sh
$ docker compose build
```

2. container run
```sh
$ docker compose up
```

3. container access
```sh
$ docker compose -it ruby_boot ash
```

4. exec ruby or irb
```sh
# ruby hoge.rb
```
or
```sh
# irb
```

Good Luck!
